# Home-Server-Lab
A version-controlled documentation of my Ubuntu-based home lab, featuring Docker-containerized media services (Plex/Jellyfin) and secure remote access via Tailscale.

## Overview
This project serves as a live "Runbook" for my home infrastructure. It manages personal media libraries (Plex/Jellyfin) and provides a secure playground for my IT and AWS certification studies.

## Tech Stack
* **OS:** Ubuntu Server 24.04.4 LTS
* **Containerization:** Docker & Docker Compose (V2)
* **Networking:** Tailscale Mesh VPN
* **Hardware:**
  * Model: Beelink U59 Mini PC (U-Series)
  * CPU: Intel Celeron N5105 (4) @ 2.900
  * GPU: Intel JasperLake [UHD Graphics]
  * Memory: 16 GB (Aprox. 15,773 MiB)
